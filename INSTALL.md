# 茶色素智能检测系统 - 安装与使用指南

## 📦 项目概述

本项目是一个基于机器学习和计算机视觉的茶色素含量检测系统，可以通过分析茶汤照片来精确预测茶黄素（TF）、茶红素（TR）、茶褐素（TB）的含量。

**技术特点：**
- 🔬 **科学依据**：基于国际标准草案IST-TP-001和行业标准NY/T 3675-2020
- 🤖 **智能预测**：采用支持向量回归（SVR）机器学习模型
- 🎨 **多维特征**：提取RGB/HSV/L*a*b*九种颜色特征变量
- 🎯 **高精度**：预测集相关系数Rp > 0.96，RPD值 > 3.50
- 💻 **桌面应用**：基于PyQt5构建专业GUI界面
- 📊 **可视化展示**：集成matplotlib实时显示检测结果

## 💡 使用说明

### 第一步：打开图片

1. 点击左侧 **"打开图片"** 按钮
2. 在文件选择对话框中选择茶汤照片
   - 支持格式：JPG, JPEG, PNG, BMP, TIFF
   - 建议分辨率：800×600 或更高
3. 图片将显示在中间的 **"原始图像"** 标签页中

### 第二步：设置参数（可选）

**预处理参数：**
- **图像增强**：选择 "无"、"直方图均衡化"、"CLAHE"、"自适应直方图"
- **滤波方法**：选择 "无"、"高斯滤波"、"中值滤波"、"双边滤波"
- **颜色空间**：选择主要颜色空间（HSV/L*a*b*/RGB）

**检测参数：**
- **置信度阈值**：调整滑块设置结果过滤阈值（0-100%）
- **批量检测数量**：设置批量处理时的图片数量

**颜色校准：**
- **启用颜色校准**：勾选后可使用标准色卡提高检测精度
- **选择标准色卡**：加载标准色卡图片
- **执行校准**：执行颜色校正计算

### 第三步：开始检测

1. 点击 **"开始检测"** 按钮
2. 系统将自动：
   - 提取图像特征
   - 执行模型预测
   - 显示处理进度
3. 检测完成后：
   - 结果将显示在右侧面板
   - 处理后图像显示在 **"处理后图像"** 标签页
   - 结果图表自动更新

### 第四步：查看结果

**数值结果：**
- 茶黄素（TF）含量
- 茶红素（TR）含量
- 茶褐素（TB）含量
- 总色素含量

**可视化图表：**
- 实时柱状图显示三种色素含量
- 柱状图上方显示预测值和置信度

**详细信息表格：**
- 检测时间
- 图像尺寸
- 文件大小
- 各色素置信度
- 平均置信度
- 预处理方法
- 校准状态

### 第五步：保存结果

**保存单次结果：**
1. 点击 **"保存结果"** 按钮
2. 选择保存路径和文件格式
3. 系统将保存检测结果到CSV/TXT/JSON文件

**导出批量结果：**
- 点击 **"导出批量结果"** 按钮可导出多张图片的检测结果（功能开发中）

## ⚙️ 配置说明

### 模型加载

程序启动时会自动尝试加载训练好的模型：

```python
# models/ 目录结构（需要提前创建）
tea_pigment_detector/
└── models/
    ├── tf_model.pkl           # 茶黄素预测模型
    ├── tr_model.pkl           # 茶红素预测模型
    ├── tb_model.pkl           # 茶褐素预测模型
    ├── scaler.pkl             # 特征归一化器
    └── feature_names.pkl     # 特征名称列表
```

**注意：** 如果模型文件不存在，程序将使用默认参数模式，预测准确性会降低。

### 校准设置

对于需要高精度检测的场景，建议：

1. 使用标准光源（如环形LED灯，色温5000K）
2. 放置标准色卡（如X-Rite ColorChecker）
3. 启用颜色校准功能
4. 在拍摄时保持相同的光照条件

## 🎯 技术原理

### 颜色特征提取

系统提取茶汤图像的9种颜色特征：

| 颜色空间 | 特征变量 | 说明 |
|---------|---------|------|
| RGB     | R, G, B | 红绿蓝三通道均值 |
| HSV     | H, S, V | 色调、饱和度、明度 |
| L*a*b*  | L, a*, b* | 亮度、红绿轴、黄蓝轴 |

### 预测模型

采用支持向量回归（SVR）算法，流程如下：

```
图像输入 → 特征提取 → Z-score标准化 → PCA降维 → SVR预测 → 结果输出
```

**模型性能指标：**
- 茶黄素：Rp = 0.983, RPD = 4.833
- 茶红素：Rp = 0.944, RPD = 3.049
- 茶褐素：Rp = 0.978, RPD = 4.501

## 🔧 故障排除

### 常见问题

**1. 程序无法启动**
- 检查Python版本是否为3.8+
- 确认所有依赖包已正确安装
- 尝试重新安装PyQt5：`pip install --upgrade PyQt5`

**2. 图片无法加载**
- 检查图片格式是否支持
- 确认图片文件是否损坏
- 尝试使用其他图片编辑软件打开

**3. 检测结果不准确**
- 确保拍摄时光照条件一致
- 启用颜色校准功能
- 检查模型文件是否正确加载
- 尝试使用不同的预处理参数

**4. 界面显示异常**
- 更新显卡驱动
- 调整显示器缩放比例
- 尝试以管理员身份运行程序

## 📈 性能优化建议

### 提高检测精度

1. **标准化拍摄环境**
   - 使用标准光源
   - 固定拍摄角度和距离
   - 使用标准色卡进行校准

2. **优化图像质量**
   - 使用高分辨率图片（建议800×600以上）
   - 避免过曝或欠曝
   - 减少阴影和反光

3. **参数调优**
   - 根据茶叶类型选择合适的颜色空间
   - 调整置信度阈值过滤低质量结果
   - 启用图像增强和滤波

### 提高处理速度

1. **批量处理**
   - 设置批量检测数量
   - 使用高分辨率图片减少处理时间
   - 关闭不必要的预处理步骤

2. **硬件升级**
   - 增加内存至8GB+
   - 使用SSD硬盘
   - 多核CPU提升并行处理能力
